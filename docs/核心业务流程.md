## 核心业务流程
### 用户注册登录流程

``` mermaid
sequenceDiagram
    participant U as 用户
    participant API as 用户服务
    participant DB as MySQL
    participant Redis as Redis

    U->>API: 1. 提交注册信息
    API->>DB: 2. 保存用户数据
    DB-->>API: 3. 返回用户 ID
    API->>Redis: 4. 生成 Token
    Redis-->>API: 5. 返回 Token
    API-->>U: 6. 返回 Token + 用户信息
```

### 创建和加入直播间流程

``` mermaid
sequenceDiagram
    participant U as 用户
    participant LS as 直播服务
    participant DB as MySQL
    participant WS as WebSocket

    U->>LS: 1. 创建直播间
    LS->>DB: 2. 保存直播间信息
    DB-->>LS: 3. 返回房间号
    LS-->>U: 4. 返回直播间详情

    U->>LS: 5. 加入直播间
    LS->>DB: 6. 更新观看人数
    LS->>WS: 7. 建立 WebSocket 连接
    WS-->>U: 8. 接收实时消息

```

### 消息发送推送流程

``` mermaid
sequenceDiagram
    participant U1 as 用户 A
    participant MS as 消息服务
    participant WS as WebSocket
    participant U2 as 用户 B

    U1->>MS: 1. 发送消息
    MS->>MS: 2. 保存消息到 DB
    MS->>WS: 3. 推送消息
    WS-->>U2: 4. 接收消息展示
```
